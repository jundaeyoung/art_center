## 🖥️ 프로젝트 소개 
누구나 쉽게 공연을 주최할 수 있고 공연을 즐길 수 있는 기회를 제공하고 싶어서 기획한 
<br>공연 대관 예약 시스템 웹사이트 프로젝트 입니다. 

## ⏰ 프로젝트 기간
- 23.05.15일 - 23.06.07일 (총 24일)
